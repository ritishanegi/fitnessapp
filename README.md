# Simple Fitness & Calorie Tracker App

## Overview
The Simple Fitness & Calorie Tracker App is an Android-based application designed to help users track their daily calorie intake and workout activities manually. The app allows users to enter meals, calculate calories consumed, record workouts, and view a daily calorie summary.

The application also includes a BMI calculator to help users understand their fitness level. The app is beginner-friendly, lightweight, and easy to use.

---

## Features

### 1. Calorie Tracking
- Add meal name
- Enter calorie value
- Store daily meal records

### 2. Workout Tracking
- Select workout type (Walking, Running, Cycling)
- Enter workout duration
- Automatically calculate calories burned

### 3. Daily Calorie Summary
- Total calories consumed
- Total calories burned
- Remaining calories

### 4. BMI Calculator
- Input height and weight
- Display BMI result

---

## Technologies Used
- Android Studio
- Java Programming Language
- XML Layouts
- Activities and Intents
- RecyclerView
- Material Design Components

---

## Objective
The objective of this project is to provide users with a simple and efficient way to monitor fitness activities and calorie intake without the complexity of advanced fitness applications.
